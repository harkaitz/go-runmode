GO-RUNMODE
==========

Decide running mode (release/debug) from an environment variable.

## Go documentation

    package runmode // import "github.com/harkaitz/go-runmode"
    
    var ReleaseMode bool = false
    func IsDebugMode() bool
    func IsReleaseMode() bool
